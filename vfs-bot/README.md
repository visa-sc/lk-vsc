# VFS-монитор слотов (Франция · Москва · туристическая виза тип C)

Следит за появлением свободных слотов записи во французский визовый центр (Москва)
и при находке шлёт письмо Насте Плинер (`anastasia.p@visa-sc.ru`), чтобы она
вручную завершила запись в кабинете клиента.

## ⚠️ ГЛАВНОЕ ПРО БЕЗОПАСНОСТЬ — почему НЕ на проде VOYO
Монитор использует headless-браузер (Chromium через Playwright). Это тяжёлый
процесс (CPU + 300 МБ–1 ГБ RAM на инстанс) с обходом антибота. **Запускать его на
том же сервере, где крутится клиентский ЛК VOYO (89.108.88.59), НЕЛЬЗЯ** — он
может «съесть» ресурсы и положить/замедлить клиентский ЛК.

→ Поэтому монитор живёт **на ОТДЕЛЬНОЙ машине** (маленький VPS). Он никак не
подключён к `server.js`, процесс `voyo` его не запускает. На клиентский ЛК он
не влияет от слова совсем (другая машина).

## Что готово, что нет
- ✅ Готово: цикл опроса раз в 2–3 мин (со случайным джиттером), дедуп уведомлений
  (`state.json`), конфиг клиентов (`config.json`), письмо Насте при найденном слоте,
  бэкофф на ошибках.
- ⏳ Нужен живой этап (помечено `TODO` в `monitor.js`): сам вход в кабинет VFS +
  обход DataDome-капчи (через 2Captcha + резидентный прокси) + парсинг доступных дат.
  Это делается с реальным доступом — селекторы/шаги VFS нельзя замаппить «вслепую».

## Что нужно, чтобы запустить
1. **Отдельный VPS** (1–2 ГБ RAM хватит). НЕ прод VOYO.
2. **2Captcha** аккаунт + ключ (для DataDome ~$3 за 1000 решений → центы/день).
3. **Резидентный прокси** (DataDome-решение без прокси не работает).
4. `cp config.example.json config.json` и вписать логин/пароль VFS клиента + окно дат.
5. `.env` со SMTP (как у VOYO): `SMTP_HOST/SMTP_PORT/SMTP_USER/SMTP_PASS/SMTP_FROM`,
   плюс `TWOCAPTCHA_KEY`, `PROXY_URL`.
6. `npm install` **внутри этой папки** (не в корне VOYO!), затем `npx playwright install chromium`.
7. `node monitor.js` (или под pm2 как ОТДЕЛЬНОЕ приложение на этой машине).

## Реалистично об ожиданиях
У VFS сильный антибот (DataDome). Даже с 2Captcha + прокси возможны блокировки —
это «гонка щита и меча», нужна обкатка. Опрос раз в 2–3 мин — щадящий режим.
Тестируем на аккаунте Дарьи Потаповой (по согласованию).
